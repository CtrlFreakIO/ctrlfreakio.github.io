I conduct laboratory classes at the **Institute of Robotics and Machine Intelligence**, 
Poznań University of Technology, within the Department of Control and Industrial 
Electronics. Classes are delivered in both full-time and part-time modes, for students 
of **Automatic Control and Robotics**.

Classes are delivered in English and Polish for full-time and part-time students, with selected 
exercises available in English for international students.

## Programmable Logic Controllers

A laboratory course covering PLC programming from foundational concepts to 
intermediate industrial applications, using **Siemens SIMATIC S7-1200** in 
the **TIA Portal** environment.

**Topics covered:**
- Basic logic functions, binary operations and program structure
- Sequential control - step-based programming and state machine design
- Timer and counter applications in process automation
- Analog inputs and outputs - signal conditioning, scaling, and processing
- Motion control using the **S7-1200 Motion Control** library - axis configuration, 
homing, jogging, and point-to-point positioning
- Closed-loop controller design - implementation of P, PI and PID regulators 
directly in PLC logic

The course bridges the gap between relay-based control covered in lower-level 
courses and the full industrial automation scenarios explored in the advanced PLC 
laboratory. The regulator design exercises provide practical context for 
understanding control theory - students implement and tune controllers on real 
physical processes rather than purely in simulation.

## Selected Applications of Programmable Controllers

An advanced laboratory course covering industrial PLC programming using **Siemens 
SIMATIC S7-1200** in the **TIA Portal** environment. The course is structured as a 
progressive sequence of exercises, building from basic controller logic to complete 
industrial automation scenarios.

**Topics covered:**
- Basic logic functions and program structure (LAD and SCL languages)
- Start-Stop circuits, edge detection, RS/SR flip-flops
- Timer and counter blocks (TON, TOF, TONR, TP, CTU, CTD, CTUD)
- Sequential control programming — conveyor automation with manual and automatic modes
- Alarm handling and emergency stop function implementation
- HMI panel configuration and visualization design (KTP600, TIA Portal Wizard)
- PLC-to-PLC Ethernet communication using PUT/GET blocks
- Remote and local control modes, data exchange between stations

This course directly reflects industrial practice - exercises are based on real 
automation scenarios including machine sequencing, diagnostic signaling, and 
operator interface design. The emergency stop exercise introduces students to 
safety-related control concepts that form the basis of my research into functional 
safety systems.


## Laboratory of Automation Elements and Devices

A foundational laboratory course introducing students to the hardware of industrial 
automation. Students work with physical laboratory stations equipped with
PLC controllers, pneumatic actuators, sensors, and relay circuits.

**Exercise stations include:**
- **Relay logic** - wiring and programming relay-based circuits: AND/OR/XOR gates, 
SET-RESET circuits, time-delayed switching
- **Pneumatic systems** - connecting and controlling single and double-acting cylinders, 
NAND logic using pneumatic elements, timed sequences
- **Tank system** - water level control using float sensors and analog input, PWM pump 
control, hysteresis controller implementation
- **Sorting line** - conveyor belt control, inductive and reflective sensor-based 
sorting, pneumatic actuator sequencing, fault handling
- **Motor control station** - PWM speed control, direction reversal, sawtooth speed 
profile generation

The course builds practical intuition for automation hardware - the same sensors, 
actuators, and relay elements that appear in real machine safety systems and that 
are the subject of my research on EN ISO 13849-1 safety category assessment.
